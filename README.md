# Multi Platform Product Sync

# Project Overview
This project is designed to help sellers upload and manage their products and directly publish them to multiple e-commerce platforms like Amazon and Flipkart.

# Features
- Product upload and management system
- Integration with Amazon marketplace
- Integration with Flipkart marketplace
- Automated product listing
- User-friendly dashboard
- API-based data sync

# Technologies Used
- PHP (Backend)
- MySQL (Database)
- JavaScript (Frontend)
- AJAX
- REST APIs

# Purpose
To simplify the process of listing products on multiple platforms and save time for sellers by automating product uploads.

# Project Setup
1. Clone the repository
2. Import database
3. Configure API keys for Amazon and Flipkart
4. Run project on local server (XAMPP/LAMP)
